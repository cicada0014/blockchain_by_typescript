# blockchain_by_typescript
타입스크립트로 블록체인을 구현해보았습니다. 

inversify를 통해 defendency를 관리하는 express 서버입니다.

## 시작 방법 

- 명령어 : PORT=xxxx npm run start 
- 기본 포트는 3005로 설정되어 있습니다.
